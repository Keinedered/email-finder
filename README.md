Скрипт поиска email-адресов на веб-странице

Описание

JavaScript скрипт для поиска и анализа email-адресов на любой веб-странице. Скрипт использует регулярные выражения с множеством специальных символов для поиска email-адресов в различных местах страницы.

 Задача скрипта

Что ищем: Email-адреса на веб-странице

Где ищем:
- В текстовом содержимом страницы
- В атрибутах HTML элементов
- В mailto: ссылках

Как обрабатываем:
- Извлекаем все уникальные email-адреса
- Группируем их по доменам
- Подсчитываем статистику (общее количество, источники)
- Выводим результаты в консоль браузера

Регулярные выражения

Скрипт использует два регулярных выражения с множеством специальных символов:

1. Основное регулярное выражение для текста:
   - Использует: `^`, `$`, `[]`, `+`, `*`, `?`, `{}`, `|`, `\.`, `\w`, `\d`, группы захвата `()`, негативные просмотры `(?:...)`

2. Регулярное выражение для mailto: ссылок:
   - Аналогичные специальные символы для поиска в атрибутах href

Использование

Скопируйте содержимое `email-finder.js` в консоль браузера на любой странице.


Демонстрация

Откройте файл `demo.html` в браузере и проверьте консоль для просмотра результатов.
